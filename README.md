## Usage:-

- Clone my repository.
- Open CMD in working directory.
- Run 
``` commandline
  pip install -r requirements.txt
 ```
 for Image Detection 
```commandline
image_op.py
``` 
for Video Detection
```commandline
video-cam.py
```
- `Human_Detection.py` contains main function for Human Detection.
